## 4. 다음 단계 권장 작업 순서

### Phase A: 기반 완성 (1주)

1. **DB 마이그레이션 실행**
   ```bash
   cd backend
   alembic revision --autogenerate -m "initial"
   alembic upgrade head
   ```

2. **초기 데이터 시드 스크립트**
   - Admin 계정 생성
   - 기본 정책 번들/임계값/플랜 생성

3. **프론트엔드 API 연동**
   - API 클라이언트 설정 (axios/fetch)
   - React Query 뮤테이션/쿼리 구현

### Phase B: 핵심 기능 테스트 (1주)

1. **AI 서비스 연동 테스트**
   - OpenAI API 키 설정 및 테스트
   - STT → LLM → TTS 파이프라인 테스트

2. **MQTT 워커 실행 테스트**
   - Docker Compose로 MQTT 브로커 시작
   - 이벤트 수신 및 처리 테스트

3. **테스트 코드 작성**
   - 핵심 API 테스트 (인증, 사용자, 케이스)

### Phase C: 모바일 앱 (2주)

1. **Flutter 프로젝트 생성**
2. **보호자 앱 핵심 화면 구현**
3. **FCM 푸시 알림 연동**

### Phase D: 디바이스 (2주)

1. **ESP32-S3 기본 펌웨어**
2. **MQTT 통신 구현**
3. **센서 연동 (IMU, 마이크)**
